# أهلاً بك، أنا أبو بكر كمال 👋

**مطور حلول تقنية شغوف بالأتمتة والأنظمة مفتوحة المصدر**

أركز بشكل أساسي على بناء بنية تحتية رقمية خالية من المشتتات. أمتلك خبرة في إدارة أنظمة **Linux** (خاصة Fedora و Ubuntu Server)، وأسعى لتطوير أدوات برمجية تسهل استهلاك المحتوى والتعامل مع البيانات بكفاءة عالية.

### 🚀 اهتماماتي التقنية:
*   **إدارة الأنظمة:** الاستضافة الذاتية (Self-hosting) باستخدام Docker و Docker Compose.
*   **الأتمتة:** تطوير روبوتات Telegram وسلاسل معالجة البيانات باستخدام Python و n8n.
*   **تطوير الويب:** بناء واجهات مستخدم سريعة وخفيفة عبر React و PWA.

### 🛠️ المشروع الحالي:
أعمل حالياً على تطوير **مشروع ركيز (Rakeez)**، وهو نظام أتمتة متكامل لمعالجة وسائط الفيديو وفصل المحتوى عن منصات التشتت التقليدية.

### 🎓 لمحة أكاديمية:
طالب في المرحلة الثانوية بـ "ثانوية عبد العزيز الخويطر"، أستعد حالياً لاختبارات التحصيلي و IELTS للتوجه نحو مسار علوم الحاسب.
